HTML interview questions

1.What is the use of the <"form"> tag?
The <"form"> tag is used to collect user input and send it to a server for processing.
It defines an HTML form that can contain input fields like text boxes, radio buttons, checkboxes, etc.
Example:<'form action="/submit" method="post">
<'/form'>


2. What is the use of the <'input> tag?

<'input> tag is used to create interactive fields for users to enter data.
It supports different type attributes like:text, password, email, number, checkbox, radio, etc.
Example:<'input type="text" name="username">


3. What is the use of the <,textarea> tag?

textarea tag is used to create a multi-line text input field.
It's useful for getting longer input from users, like comments or messages.
Example: <'textarea name='message' rows='4' cols='30'><'/textarea>


4. What is the use of the <'button> tag?

The button tag is used to create a clickable button.
It can be used to submit a form, reset it, or trigger JavaScript actions.
Examples:- Submit: <'button type='submit'>Submit<'/button>- JS Action: <'button onclick="alert('Hello!')">Click Me<'/button
